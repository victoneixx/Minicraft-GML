Esse é um projeito feito com base no jogo minicraft, feito pelo notch em 2011 para uma gamejam chamada Ludum Dera 22. 
A ideia desse projeto, e refazer o jogo no gamemaker e testar as minhas habilidades de programação que eu aprendi durante 7 meses de estudo.
Faço um curso chamando NoneClass, onde eu aprendo a como programar jogos no gamemaker.

Você pode baixar o SocerCode do jogo para ver como que eu fez certas mecanicas e códigos para fazer o jogo.
O projeto é bem basico, mias pode ter algo que você possa aprender de novo vendo o código.
Fique avontade e explore o código =D
